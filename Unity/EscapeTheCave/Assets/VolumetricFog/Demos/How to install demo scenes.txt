To install and execute the demo scenes:

1) Import Unity Standard Assets package (ONLY REQUIRED FOR DEMO SCENES)
   (You can download it from the Asset Store)
2) Double click on Demos package inside the Demos folder to import them
3) The demos are now available inside Demos folder.

NOTE: Demo scenes are provided to let you familiarize quickly with the main asset features. When importing the package in your real project, you don't need to import demo folder at all.
